\subsection{Detailed Description}
Represents an entire background register for the display with simple functions for easy access to contained information. 

\begin{DoxyWarning}{Warning}
Name may suggest this is R\-E\-G\-I\-S\-T\-E\-R, it is simply a struct. It replaces direct register access, and so maintains the naming conventions previously used there from A.\-Sampson's \hyperlink{gba_8h_source}{gba.\-h} library. 
\end{DoxyWarning}
