Parameters:
bgNumberA number between 0 and 3, for which background layer this is
xOffThe starting horizontal offset to the left

Warning:
bgNumbers out of the 0->3 range are placed at the ends of this range to avoid running off memory

Initialises this BG, with specified charblock and screenblock numbering.

Parameters:
bgNumberA number between 0 and 3, for which background layer this is
xOffThe starting horizontal offset to the left
yOffThe starting vertical offset, downwards
charblockthe charblock to be used for this background
screenblockthe screenblock to be used for this background
Warning:
bgNumbers and charblocks are forced into the range 0->3, use charblock 0 first and ascend.
screenblocks are forced into the range 30->0; use screenblock 30 first and then descend.
