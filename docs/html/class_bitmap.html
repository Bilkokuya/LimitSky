Detailed Description

template<class T>
class Bitmap< T >

Bitmap Resource, comprising of non-platform specific bitmapData, and size data. On the GBA, the sizes are specifically limited to 8x8 amounts, as this is tile data.

Parameters:
TThe type of bitmapData to be used. On the GBA this is uint8_t (unsigned char), due to indexed pixels
